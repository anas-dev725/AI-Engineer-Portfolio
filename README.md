## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/anas-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd anas-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development
Start the development server:
```bash
npm run dev
```
The app will be available at `http://localhost:3000`.

### Build
To build the project for production:
```bash
npm run build
```

## 🌐 Deployment

This project is configured for deployment on **GitHub Pages**. You can deploy it using:
```bash
npm run deploy
```
